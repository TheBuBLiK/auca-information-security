## Overview

This repository tracks my course progress from Linux fundamentals to basic offensive and defensive security workflows:

- Linux file operations and scripting
- Permissions, users, and scheduled tasks
- Package management and service configuration
- Installing tools from source
- Brute-force attack simulation with Hydra
- Keylogger simulation with controlled log exfiltration to an API

Most labs include terminal transcripts (`terminal.txt`) and screenshot evidence (`.png` files).
